# 这是什么？

在AI大道当行的时代，DeepWiki提供免费的AI用以分析全球的Github仓库，只要你的仓库有一些Star，稍微流行一些，就有可能被DeepWiki收录，并且自动通过AI大模型创建专属于你的Github仓库的强大Wiki！

![](../assets/images/deepwiki.png)

# 如何查看我的仓库是否被收录？

前往 [DeepWiki | AI documentation you can talk to, for every repo](https://deepwiki.com/) 搜索你的 **Github 用户名** 即可看到当前DeepWiki收录的你的所有仓库

![](../assets/images/deepwiki-1.png)

DeepWiki爬取的频率不算很高，我们可以通过在仓库README添加一串文本来声明该仓库希望DeepWiki收录，之后，按DeepWiki所说，将会每周爬取一次最新代码并生成Wiki

```sql
[![Ask DeepWiki](https://deepwiki.com/badge.svg)](https://deepwiki.com/你的Github用户名/你的Github仓库)
```

![](../assets/images/deepwiki-3.png)

之后，若DeepWiki已经收录该仓库，你将可以看到醒目的提示

 ![](../assets/images/deepwiki-2.png)

# 接下来...？

你可以将DeepWiki贴到你的网站，服务，App上，让用户遇到问题直接询问DeepWiki！无需再手动搭建你的知识库了！

![](../assets/images/deepwiki-5.png)

![](../assets/images/deepwiki-4.png)

又或者，你也可以打开其他仓库的Wiki，尝试通过询问DeepWiki AI来快速上手他们！
